# 使用方法&注意事项

1. 到Tag找到相应版本下载好解压包解压
2. 下载Python≧3.10版本
3. 按下`ctrl+r`,在对话框内输入:`cmd`**
4. 在cmd内输入如下代码安装或更新(50%的问题可以提供更新此库解决):
```
pip install ttkbootstrap
```
